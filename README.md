# calendar-laravel-dev

## 使用套件
- FullCalendar
- jQuery
- Bootstrap
- FontAwesome

## 如何使用

### 安裝
```
composer install
php artisan migrate
php artisan serve
```

### User 資料產生
```
php artisan db:seed
```

### 登入
```
test@gmail.com
Qq123456
```
